# TsExpressReactScaffold
An scaffold for Express + React using typescript

# Provisioning
* Install [Nodejs](https://nodejs.org/en/)
* Install [yarn](https://yarnpkg.com/en/)
    ```powershell
    npm install -g yarn
    # cd ~/path/to/project
    yarn set version berry
    ```
# Initial Project
* Develop environment: 
    > \> yarn install
* Production environment:
    > \> yarn install --prod
* Add dependency:
    > \> yarn add --dev
# Command
* Build 
    > \> npm run build
* Launch website
    > \> npm run nodemon
* Run Unit Test
    > \> npm run test
* Launch in Production
    > \> npm run forever
# Other
* Add dev dependency
    > \> yarn add xxxx -D
* Remove dependency
    > \> yarn remove xxxx
